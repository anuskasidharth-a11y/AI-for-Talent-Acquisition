# 05_Resources

This folder is my personal AI recruiting knowledge base. It is not another set of prompts or workflows. It is the thinking behind everything else in this repository: the tools I trust, the way I write prompts, the boundaries I hold, and the checkpoints I never skip.

If 01 to 04 show what I build, this folder shows why I build it that way.

## Contents

- **AI_Tools_I_Use_And_Why.md** — my actual AI stack, what each tool is good for in a recruiting context, and where each one lets me down.
- **Prompt_Engineering_Notes.md** — the framework I use to write prompts, with real examples from sourcing and screening work.
- **Recruiting_AI_Best_Practices.md** — where I use AI, where I deliberately do not, and how I handle privacy, bias, and data security.
- **AI_Limitations_And_Human_Checkpoints.md** — the failure modes I watch for and the checklist I run before AI output touches a real candidate or client.
- **Boolean_Search_CheatSheet.md** — the search fundamentals that still sit underneath everything AI has changed about sourcing.

## Currently exploring

MCP, AI agents, RAG, and vector databases, with the goal of understanding how these will change sourcing and screening workflows over the next year. I will fold what I learn back into the files above rather than starting a separate log for it.
