# Communication Prompts

Prompts for candidate and stakeholder messaging at every stage.

## Cold outreach message

```
Write a cold outreach message to a candidate for [role] at [company].
Candidate background: [paste background]
Tone: [professional/friendly/direct]
Platform: [LinkedIn/email/InMail]

Ground the message in one specific detail from their background.
Keep it under [character/word limit]. End with a specific, low
effort ask, not a vague "let me know your thoughts."
```

## Rejection message (post-interview)

```
Write a rejection message for a candidate who reached the [stage]
stage for [role]. They were not selected because: [reason, kept general
if sensitive]. Keep the tone respectful and specific enough to feel
genuine, without giving detailed competitive feedback. Under 100 words.
```

## Offer stage follow-up

```
A candidate has an offer for [role] and has gone quiet for [X days].
Write a short, low pressure follow-up message that checks in without
sounding like a countdown or an ultimatum.
```

## Hiring manager status update

```
Write a status update for a hiring manager on the [role] search.
Candidates in pipeline: [number]
Stage breakdown: [e.g. 3 in screening, 2 in interview, 1 offer stage]
Key blocker if any: [describe]

Keep it to 4 to 5 sentences, lead with the most important information.
```

## Declining a candidate's counter-offer situation

```
A candidate accepted our offer but is now being counter-offered by
their current employer. Write a message that reaffirms our offer's
value without being pushy, and gives them space to make a clear decision
by [date].
```

## Referral request to internal employee

```
Write a message to an internal employee asking if they know anyone
for a [role] opening. Include 2 to 3 lines on what the role involves
and why it is a good opportunity, keep it short enough to forward easily.
```
